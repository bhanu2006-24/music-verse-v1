Update Library

Easily manage your music using the music.json file.

1. Add Links

Open downloads.json in your project and add URLs.

{
  "videos": [
    "https://youtu.be/..."
  ],
  "playlists": [
    "https://youtube.com/playlist?list=..."
  ]
}

2. Sync Music

Run this command to download new music and update your library:

npm run sync
