# Music Verse v1

A premium music gallery and downloader application, inspired by Ani-Gallery.

## Features

- **Music Gallery**: A beautiful, glassmorphism-inspired interface to browse and play your music collection.
- **Download API**: Built-in tool to download high-quality audio from YouTube.
- **Auto-Organization**: Automatically organizes downloaded music into the `music/` folder.
- **Premium UI**: Smooth animations, dark mode, and responsive design.

## Tech Stack

- **Frontend**: HTML5, Vanilla JavaScript, Tailwind CSS (via CDN for speed/simplicity).
- **Backend**: Node.js, Express.
- **Tools**: yt-dlp (for reliable downloads).

## Setup

1. Install dependencies: `npm install`
2. Start the server: `npm start`
3. Open `http://localhost:3000`
